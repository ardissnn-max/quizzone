## 📖 Tentang Proyek

**QuizZone** adalah aplikasi kuis pengetahuan umum berbasis web yang dibangun menggunakan HTML, CSS, dan JavaScript murni (tanpa library tambahan). Proyek ini menampilkan antarmuka modern dengan tema gelap, animasi halus, dan sistem poin yang membuat pengalaman bermain lebih seru dan menantang.

Proyek ini dibuat oleh **Ardis Noer Haykal** dengan NIM **125300010** sebagai bagian dari tugas mata kuliah **Bahasa Indonesia**.

---

## ✨ Fitur

- 🧠 **10 Soal Pilihan Ganda** — Mencakup berbagai kategori (Teknologi, Sains, Geografi, Sejarah, Matematika)
- ⏱️ **Timer 15 Detik Per Soal** — Ditampilkan dengan animasi cincin yang berubah warna
- 🎯 **Sistem Skor Dinamis** — Semakin cepat menjawab, semakin banyak poin didapat
- ✅❌ **Feedback Langsung** — Jawaban benar/salah langsung ditampilkan dengan warna
- 📊 **Layar Hasil Akhir** — Menampilkan total skor, jumlah benar, salah, dan akurasi
- 🔁 **Fitur Ulangi Kuis** — Bisa bermain ulang tanpa me-refresh halaman
- 📱 **Responsif** — Tampilan menyesuaikan ukuran layar perangkat

## 🎮 Cara Bermain

1. Buka `index.html` di browser
2. Klik tombol **"Mulai Kuis →"**
3. Baca pertanyaan dan pilih jawaban sebelum timer habis
4. Skor dihitung berdasarkan kecepatan menjawab
5. Setelah 10 soal selesai, lihat hasil akhirmu
6. Klik **"Main Lagi"** untuk mengulang kuis
